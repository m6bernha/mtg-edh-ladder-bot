# Assets

## `edh-ladder-icon.jpg`

The bot's avatar / app icon — a five-colour arcane spark over crossed swords, on a
dark field. 768×768 JPEG. Original artwork (generated), no third-party IP, safe to
ship in a public MIT repo.

**To use it as the bot's avatar**, upload it by hand in the
[Discord Developer Portal](https://discord.com/developers/applications):

- **App icon:** your application → **General Information** → **App Icon**.
- **Bot avatar:** your application → **Bot** → **Icon** (this is what shows next to
  the bot's messages).

Discord accepts PNG/JPG and wants at least 512×512, so this is fine as-is. Swap in
your own art any time — nothing in the code references this file.
